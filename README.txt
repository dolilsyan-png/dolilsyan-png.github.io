ВАРИАНТ 1: GITHUB PAGES (рекомендуется, бесплатно)
═══════════════════════════════════════════════════

1. Зарегистрируйся на https://github.com
2. Нажми "+" → "New repository"
3. Название: vocab (или любое другое)
4. Поставь галочку "Public"
5. Нажми "Create repository"
6. Нажми "uploading an existing file"
7. Перетащи ВСЕ 5 файлов из этой папки
8. Нажми "Commit changes"
9. Зайди в Settings → Pages
10. Source: "Deploy from a branch", Branch: main, папка: / (root)
11. Нажми Save
12. Через 1-2 минуты появится ссылка:
    https://ИМЯ_ПОЛЬЗОВАТЕЛЯ.github.io/vocab/vocabulary.html

НА ТЕЛЕФОНЕ:
13. Открой эту ссылку в Chrome на Android
14. Нажми три точки → "Добавить на главный экран"
    ИЛИ появится баннер "Установить приложение" — нажми его
15. Готово! Иконка появится на рабочем столе

═══════════════════════════════════════════════════
ВАРИАНТ 2: ПРОСТО КАК ФАЙЛ (без установки)
═══════════════════════════════════════════════════

1. Перекинь vocabulary.html на телефон
   (через Telegram себе, Google Drive или кабель)
2. Открой файл в Chrome
3. Три точки → "Добавить на главный экран"
   (откроется как вкладка браузера, не как приложение)

ПЕРЕНОС ДАННЫХ МЕЖДУ УСТРОЙСТВАМИ
═══════════════════════════════════════════════════
